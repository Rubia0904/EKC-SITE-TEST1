一、策略概述

本文件依據智慧建築評估手冊，針對電動車輛及其他充電裝置相關的評估項目，提供完整的得分策略與實施建議。充電裝置相關評估項目橫跨「基礎設施指標」與「安全防災指標」，總計可獲得 7 分。

二、評估項目與配分

2.1 基礎設施指標（2分）

項目 1.2.4：設置電動車輛充電設備電能維運管理系統

評估項目 配分 評估內容
電動車輛充電設備電能維運管理系統 2 分 停車空間導入電動車輛電能管理系統
📌 評估目的：
為有效落實電動車輛推動，擴大節能減排之效益，鼓勵建物提供安全、充裕、普及的充電設備及裝置，並導入電動車輛電能管理系統，以確保用電安全、後續擴充彈性及維運。

2.2 安全防災指標（5分）

項目 3.1.1.3：用電異常偵測

評估項目 配分 評估內容
充電裝置用電異常偵測 2 分 用電異常訊號整合至智慧建築管理系統

評估內容：
對於長時間使用的用電設備或充電裝置（如：汽車充電、機車充電電力線路或長時間用電設備等）具有用電異常偵測功能，並將訊號整合至智慧建築管理平台。

項目 3.1.3.3：用電異常自動斷電

評估項目 配分 評估內容
充電裝置用電異常自動斷電 3 分 具備用電異常自動斷電功能，且能將訊號整合至智慧建築管理平台

評估內容：
系統具備用電異常自動斷電功能（如：汽車充電電力線路、機車充電電力線路或常時用電設備等），並將訊號整合至智慧建築管理平台。

三、得分策略與實施建議

3.1 完整得分路徑（總計 7 分）

基礎設施指標（2分）+ 安全防災指標（5分）= 總計 7 分
├─ 1.2.4 電動車輛充電設備電能維運管理系統（2分）
└─ 安全防災
   ├─ 3.1.1.3 用電異常偵測（2分）
   └─ 3.1.3.3 用電異常自動斷電（3分）

3.2 分階段實施策略

第一階段：基礎建置（2分）

🎯 目標： 建立充電設備電能維運管理系統

實施要點：

  1. 規劃停車空間充電設施配置
  2. 導入電動車輛電能管理系統
  3. 確保用電安全及後續擴充彈性
  4. 建立維運管理機制

候選證書階段送審資料：

  • 電動車輛電能維運管理系統架構圖
  • 電力單線圖
  • 充電設備規格及配置說明

智慧建築標章階段送審資料：

  • 現勘確認電能維運管理系統畫面
  • 竣工圖說
  • 系統操作手冊

第二階段：安全監控（+2分，累計4分）

🎯 目標： 建立用電異常偵測機制

實施要點：

  1. 於充電裝置安裝用電異常偵測設備
  2. 建立長時間用電設備監控機制
  3. 整合訊號至智慧建築管理平台
  4. 確保即時監測與記錄功能

候選證書階段送審資料：

  • 電力迴路單線圖
  • 充電裝置設備或插座配置圖
  • 用電管理功能規範
  • 智慧建築管理平台架構圖
  • 智慧化功能 I/O 點數表

智慧建築標章階段送審資料：

  • 電力迴路單線圖（竣工）
  • 充電裝置設備或插座配置竣工圖說
  • 系統畫面及佐證照片
  • 智慧建築管理平台架構竣工圖說
  • 智慧化功能 I/O 點數表（竣工）

第三階段：自動防護（+3分，累計7分）

  3. 整合連動控制至智慧建築管理平台
  4. 建立事件記錄與通知機制
  5. 確保人員安全與設備保護

  • 系統連動邏輯圖

4.2 用電異常偵測標準

建議建立以下異常偵測機制：

異常類型 偵測指標 建議閾值 處理方式
過電流 充電電流 > 額定電流 120% 告警 + 自動斷電
欠電壓 充電電壓 < 額定電壓 85% 告警
過電壓 充電電壓 > 額定電壓 110% 告警 + 自動斷電
溫度異常 設備溫度 > 60°C 告警 + 自動斷電
漏電 漏電流 > 30mA 告警 + 自動斷電
長時間使用 連續充電時間 > 8 小時 告警

  • 溫度值讀取

  • 斷電器狀態

輸出點（Output）：

  • 充電啟動/停止控制
  • 自動斷電控制

💡 建議每個充電點位至少配置：
• 輸入點：6-8 點
• 輸出點：4-6 點

六、常見問題與解決方案

Q1：充電裝置是否需要全部納入管理系統？

A： 是的。所有充電裝置（包括汽車充電樁、機車充電插座）都應納入電能維運管理系統，並具備用電異常偵測與自動斷電功能，才能獲得完整分數。

Q2：既有建築物如何改善以符合評估標準？

A： 建議採取以下步驟：

  1. 盤點現有充電設施數量與位置
  2. 評估現有電力系統容量
  3. 增設用電監測與控制設備
  4. 建立或擴充智慧建築管理平台
  5. 整合既有與新增設備至統一平台

Q3：用電異常自動斷電是否會影響使用者體驗？

A： 妥善設計可以兼顧安全與體驗：

  1. 設定合理的異常判斷閾值
  2. 提供多層次告警機制（輕微異常先告警，嚴重異常才斷電）
  3. 建立即時通知機制（APP推播、簡訊通知）
  4. 提供手動復歸功能
  5. 記錄異常事件供後續分析改善

Q4：是否需要為每個充電點獨立配置斷電器？

A： 建議每個充電點或充電迴路獨立配置，以達到：

  1. 精確定位異常點位
  2. 避免影響其他正常使用的充電點
  3. 提升系統可靠性與維運效率
  4. 符合評估標準對「自動斷電功能」的要求

七、效益分析

7.1 安全效益

  • 預防火災風險： 即時偵測並處理用電異常，降低火災發生機率
  • 保護設備安全： 自動斷電機制保護充電設備及車輛
  • 人員安全保障： 降低觸電風險，提升使用安全性

7.2 管理效益

  • 集中監控管理： 透過統一平台管理所有充電設施
  • 維運效率提升： 即時掌握設備狀態，快速處理異常
  • 數據分析應用： 累積用電數據，優化充電設施配置

7.3 節能效益

  • 用電行為分析： 掌握充電尖離峰時段，進行負載管理
  • 能源成本優化： 透過數據分析制定合理充電策略
  • 擴大減碳效益： 推動電動車使用，降低碳排放

十、結論與建議

10.1 核心建議

  1. 優先規劃： 於建築規劃階段即納入充電設施智慧化考量
  2. 系統整合： 充電設施應與智慧建築管理平台完整整合
  3. 安全為重： 用電異常偵測與自動斷電為安全防災核心功能
  4. 彈性擴充： 預留未來擴充空間，因應電動車普及趨勢
  5. 數據應用： 善用充電數據進行能源管理與優化

10.2 未來趨勢

  • 智慧充電調度： 結合 AI 進行充電排程與負載管理
  • V2G 技術整合： 車輛對電網（Vehicle-to-Grid）雙向充放電
  • 再生能源結合： 整合太陽能光電等再生能源
  • 碳權管理： 充電數據納入碳足跡管理系統

10.3 投資效益評估

初期投資：

  • 充電設備及管理系統：建議預算約為總工程款 0.5-1%
  • 智慧化控制設備：每充電點約增加 15-20% 成本

長期效益：

  • 降低火災風險及保險成本
  • 提升建築物智慧化等級與價值
  • 吸引重視環保的租戶/買家
  • 符合政府淨零碳排政策方向
